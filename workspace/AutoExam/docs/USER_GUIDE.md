# AutoExam 用户操作指南

## 系统简介

AutoExam是一个自动驾驶高危场景智能生成与测试系统，专注于无保护左转场景的生成、测试和分析。系统采用对抗性生成策略，能够智能地发现Apollo自动驾驶系统的"攻击面"。

## 核心功能

### 1. 场景生成
- **随机生成**: 基于8个参数维度生成多样化场景
- **对抗性生成**: 在高危参数空间内密集采样
- **自然语言生成**: 支持自然语言描述生成场景

### 2. 场景测试
- **批量测试**: 支持批量场景测试
- **结果分析**: 自动分析测试结果
- **失败聚类**: 智能聚类失败模式

### 3. 报告生成
- **性能分析**: Apollo系统表现分析
- **失败模式**: 失败案例聚类分析
- **改进建议**: 针对性的改进建议

## 8个参数维度

| 参数 | 取值范围 | 说明 |
|------|----------|------|
| 自车速度 | 5-30 km/h | 接近路口的速度 |
| 对向车速 | 30-80 km/h | 对向直行车的速度 |
| 时间间隙 | 2-8秒 | 自车与对向车到达冲突点的时间差 |
| 对向车型 | 轿车/卡车/公交车 | 不同车型影响博弈行为 |
| 天气 | 晴/雨/雾/夜 | 影响感知难度 |
| 视野遮挡 | 有/无 | 对向大车遮挡后面小车 |
| 交通流量 | 低/中/高 | 多车道多辆车流 |
| 行人/非机动车 | 有/无 | 增加复杂度 |

## 使用流程

### 方式一：Web界面操作（推荐）

#### 1. 启动系统
```bash
cd /home/xcc/aiX-platform/workspace/AutoExam
python ui/app.py
```

#### 2. 访问Web界面
打开浏览器访问: http://localhost:5000

#### 3. 无保护左转场景操作流程

**步骤1: 生成场景**
- 进入"无保护左转"页面
- 选择生成模式：
  - **快速生成**: 输入数量，选择难度，一键生成
  - **自然语言**: 输入描述，如"生成100个雨天卡车遮挡场景"
  - **对抗性生成**: 基于已有测试结果智能生成高危场景

**步骤2: 查看场景库**
- 进入"场景管理"页面
- 查看所有生成的场景

- 可导出OpenSCENARIO格式

**步骤3: 执行测试**
- 选择要测试的场景
- 点击"执行测试"按钮
- 等待测试完成（需要CARLA环境）

**步骤4: 查看结果**
- 进入"测试结果"页面
- 查看成功率、碰撞率、超时率
- 查看失败案例分析

**步骤5: 生成报告**
- 点击"生成测试报告"
- 下载完整的测试报告（Markdown格式）

## 对抗性生成流程

### 核心闭环逻辑

```
第0轮: 初始随机生成
  └─ 生成500个随机场景
  └─ 执行测试
  └─ 分析失败模式

第1轮: 对抗性聚焦
  └─ 基于失败分析，在高危参数空间生成250个场景
  └─ 例如: 对向车速>60km/h + 时间间隙<3秒 + 雨天
  └─ 执行测试
  └─ 分析更细分的失败模式

第2轮: 深度挖掘
  └─ 基于第1轮结果，进一步聚焦
  └─ 例如: 对向车速70-80km/h + 时间间隙2-2.5秒 + 雾天 + 有遮挡
  └─ 执行测试
  └─ 发现更精确的攻击面

第3轮: 精准打击
  └─ 在最细分的参数空间内生成场景
  └─ 验证高危参数组合
```

### 预期效果

- **初始随机**: 失败率约20-30%
- **第1轮对抗性**: 失败率提升至50-60%
- **第2轮对抗性**: 失败率提升至70-80%
- **第3轮对抗性**: 失败率可能达到90%+

## 常见问题

### Q1: 如何查看生成的场景文件？
A: 场景文件保存在 `./scenarios/unprotected_left_turn/` 目录下，格式为JSON。

### Q2: 如何导出OpenSCENARIO格式？
A: 在场景库页面，选择场景后点击"导出OpenSCENARIO"按钮，文件将保存到 `./openscenario/` 目录。

### Q3: 对抗性生成需要多长时间？
A: 取决于初始场景数和迭代次数。500个场景×3轮迭代，每轮测试约需1-2小时（使用CARLA仿真）。

### Q4: 如何解释高危参数组合？
A: 高危参数组合表示在这些参数条件下，Apollo系统失败的概率显著高于平均水平。例如"对向车速>60km/h + 时间间隙<3秒 + 雨天"可能表示Apollo在雨天高速接近场景下的决策算法存在缺陷。

### Q5: 可以自定义参数范围吗？
A: 可以，修改 `generators/unprotected_left_turn_generator.py` 中的 `parameter_ranges` 字典即可。
